## 🚀 Featured Projects

### 🏗️ [SQL Data Warehouse & Analytics Project](https://github.com/adetonayusuf/sql-data-warehouse-project)
> *Building a modern data warehouse with SQL Server, including ETL processes, data modelling, and analytics*

A complete end-to-end data solution built from scratch:

- 🥉 **Bronze Layer** — Raw ERP and CRM data ingested via stored procedures
- 🥈 **Silver Layer** — Full data cleansing: deduplication, NULL handling, standardisation
- 🥇 **Gold Layer** — Star schema (fact_sales + dim_customers + dim_products)
- 🔍 **EDA** — 9-section exploratory analysis: KPIs, trends, segmentation, rankings
- 📊 **Advanced Analytics** — YoY analysis, window functions, customer & product report views
